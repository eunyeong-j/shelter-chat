# shelter-chat

This chat app is a closed system that stores data in the host's local environment.

## Tools

### Client

- React.js
- Tanstack Query(react-query)
- TailwindCSS
- vite

### Server

- Node.js
- Express
- SQLite3

## Installation

```
// root
$ npm install

// cd client
$ npm install

// cd server
$ npm install
```

## Server Start

```
// root
$ npm run dev
```
